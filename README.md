# Portfolio

The personal portfolio site of Ayla Myers, made using Next.js.

## Setup

Install dependencies with `npm install -D`.

Run `npm run dev` then open [http://localhost:3000](http://localhost:3000).

## Deploy

Run the [Deploy to GitHub Pages](https://github.com/aylanonsense/aylanonsense.github.io/actions/workflows/deploy.yml) action and view the results at [https://aylanonsen.se](https://aylanonsen.se).
